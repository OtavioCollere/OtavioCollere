Desenvolvedor Backend com foco em Node.js, NestJS e TypeScript. Experiência profissional desde 2022, atuando em empresas de SaaS com sistemas em produção para mais de 1 milhão de usuários ativos.

## Projetos em Destaque

### [lumi-energy-invoices](https://github.com/OtavioCollere/lumi-energy-invoices)
API de processamento de faturas de energia via LLM multimodal. Extração de dados de PDFs com OpenAI GPT-4o, Clean Architecture (Ports & Adapters), PostgreSQL/Prisma, testes E2E isolados por schema UUID, Prometheus + Grafana e deploy automatizado no GCP Cloud Run.

`NestJS` `TypeScript` `PostgreSQL` `Prisma` `OpenAI` `Prometheus` `Vitest` `Docker` `GitHub Actions` `GCP Cloud Run`

---

### [invoice-service-microservices-](https://github.com/OtavioCollere/invoice-service-microservices-)
Plataforma de microserviços com mensageria assíncrona via RabbitMQ, API Gateway com Kong, infraestrutura provisionada na AWS (ECS Fargate, ALB, NLB, ECR) via Pulumi (IaC) e observabilidade distribuída com OpenTelemetry e Grafana Cloud.

`Node.js` `TypeScript` `Fastify` `RabbitMQ` `PostgreSQL` `Kong` `AWS ECS Fargate` `Pulumi` `OpenTelemetry` `Docker`

---

### [booking-service](https://github.com/OtavioCollere/booking-service)
Plataforma de reservas distribuída com NestJS e arquitetura de microserviços. Autenticação JWT distribuída, integração com Stripe, comunicação assíncrona via RabbitMQ e API Gateway GraphQL com Apollo Federation.

`NestJS` `TypeScript` `RabbitMQ` `Stripe` `GraphQL` `PostgreSQL` `Docker` `GCP`

---

### [reactivate-cliniker](https://github.com/OtavioCollere/reactivate-cliniker)
Agente de reativação de pacientes para clínicas de estética. Identifica inativos, gera mensagens personalizadas via OpenAI e envia via WhatsApp. Deploy com cron no Railway, deduplicação via PostgreSQL e integração com SaaS próprio.

`Python` `OpenAI` `PostgreSQL` `Pydantic` `httpx` `Railway`
